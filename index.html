Introdução

Este documento apresenta os passos para instalação do ambiente de programação do Bloco+ Bot. O ambiente de programação do Bloco+ Bot é composto por quatro partes:

  • Uma modificação do software Arduino IDE com Ardublock (interface de programação em blocos).
  • Driver de comunicação entre o Windows e a placa ESP32 (cérebro do Bloco+ Bot).
  • Arquivo de configuração do Arduino IDE para reconhecer a placa ESP32.
  • Biblioteca de códigos necessária para programar o servomotor ou reprodução de sons.

Etapa 1 - Instalação do Arduino Ardublock

  1. Baixe o arquivo "instalador-arduino-rev4-v1.exe" na seção de downloads deste site.
  2. Execute o arquivo e clique em "OK" para iniciar a instalação.
  3. Siga as instruções na tela e aguarde o final da instalação.
  4. Feche o instalador após a conclusão.

Etapa 2 – Instalação do driver

  1. Abra o programa Arduino.
  2. Conecte o Bloco+ Bot ao computador através do cabo USB.
  3. Clique em "Ferramentas" e verifique se a opção "Porta" está disponível.
  4. Se a porta pode ser selecionada (ex: COM3 ou outra porta COM > 3), o driver já está instalado - pule para a Etapa 3.
  5. Caso a porta não esteja disponível:
    1. Baixe o arquivo "driver-esp32.zip" na seção de downloads.
    2. Extraia o arquivo e abra a pasta "driver-esp32".
    3. Execute o arquivo "CP210xVCPInstaller_x64.exe".
    4. Avance pela instalação do driver até o fim.
    5. Feche o Arduino e repita os passos 1 a 3.

Etapa 3 – Instalação da placa ESP32 no Arduino

  1. Abra o Arduino.
  2. Clique em "Arquivo" > "Preferências".
  3. No campo "URLs Adicionais para Gerenciadores de Placas" insira: https://dl.espressif.com/dl/package_esp32_index.json
  4. Clique em OK.
  5. Clique em "Ferramentas" > "Placa:" > "Gerenciador de Placas".
  6. Pesquise por "esp32" e instale "esp32 by Espressif Systems".
  7. Após a instalação, selecione "ESP32 Arduino" > "ESP32 Dev Module".

Etapa 4 – Instalação da biblioteca ESP32Servo

  1. Abra o Arduino.
  2. Clique em "Sketch" > "Incluir Biblioteca" > "Gerenciar Bibliotecas".
  3. Aguarde a atualização das bibliotecas (barra azul inferior).
  4. Pesquise por "esp32servo".
  5. Instale "ESP32Servo by Kevin Harrington".
  6. Após o final da instalação, clique em "Fechar".
